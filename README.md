Sales Analytics Pipeline

A Python pipeline for cleaning, analyzing, and visualizing sales data.
This project demonstrates a structured data workflow including data validation, cleaning, business metrics, and visualizations.

Features

Data validation and quality checks

Data cleaning and preprocessing

Business metrics calculation

Automated visualizations

Configurable pipeline using JSON

Logging for monitoring pipeline execution

Tech Stack
Python
Pandas
Seaborn
Matplotlib
Logging
JSON Configuration

Pipeline Steps

The pipeline performs the following steps:

Load configuration from JSON

Initialize logging system

Load dataset

Validate dataset quality

Clean and preprocess data

Compute business metrics

Generate visualizations

Visualizations

The pipeline automatically generates charts such as:

Sales by City

Monthly Sales Trend

Top Customers

Order Amount Distribution

These charts are saved in the figures directory.

Configuration

The pipeline uses a JSON configuration file.

Example:

{
  "data_file": "data/orders.csv",
  "log_file": "logs/pipeline.log",
  "figures_path": "figures"
}
Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/sales-analytics-pipeline.git

Install dependencies:

pip install pandas seaborn matplotlib
Run the Pipeline
python pipeline.py
