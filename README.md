# Sales Analytics Pipeline

A Python-based data analytics pipeline for processing, analyzing, and visualizing sales data in a structured and automated workflow.

This project demonstrates a complete data pipeline including data validation, cleaning, transformation, business metrics calculation, and visualization generation.

---

## 🚀 Features

* Data validation and quality checks
* Data cleaning and preprocessing
* Business metrics calculation (KPIs)
* Automated data visualizations
* Configurable pipeline using JSON files
* Logging system for monitoring execution flow

---

## 🛠 Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn
* Logging
* JSON configuration

---

## 📊 Pipeline Overview

1. Load raw sales data
2. Validate data quality
3. Clean and preprocess dataset
4. Calculate business KPIs
5. Generate visual reports
6. Log execution steps

---

## 🎯 Purpose

This project was built to demonstrate real-world data engineering and analytics skills, focusing on building reusable, configurable, and maintainable data pipelines.
